# UnClutter

UnClutter started as a personal cleanup assistant built with a custom GPT agent for managing house chores, organizing spaces, and staying on top of home maintenance.

I developed it initially for my own home because I wanted an intelligent helper that could understand my cleanup habits, offer smart suggestions, and turn messy spaces into manageable routines. After using it for a while, my friends tried it and loved how practical and easy it made running cleanup sessions.

That support encouraged me to take UnClutter public so others can benefit too.

## What It Does

- Helps plan and prioritize cleanup tasks for rooms, closets, kitchens, and more
- Suggests efficient order and timing to make cleanup feel less overwhelming
- Offers personalized tips for organizing, decluttering, and maintaining a tidy home
- Supports habit-building around regular cleaning and simple home upkeep

## Why It Exists

UnClutter is about making home cleanup feel less like a chore and more like a sustainable routine. It combines the flexibility of a custom GPT agent with real-world cleaning ideas so users get helpful, human-friendly guidance.

## Files in This Repository

- `unclutter-gpt-instructions.md` and other versions: custom instruction files used to shape the GPT agent's behavior for cleanup tasks

## Get Started

Use these instruction sets to build or refine your own house cleanup assistant with a GPT-based workflow. The goal is to make cleanup easier, more consistent, and more enjoyable.
